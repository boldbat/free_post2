
## Quick Start

Want to run this locally? Here's how:

```bash
# Clone the repo
git clone https://github.com/boldbat/free_post2.git
cd free_post2

# Install dependencies
npm install

# Run development server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
